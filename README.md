# 🍬 Candy Crush Game

A simple web-based Candy Crush game built using HTML, CSS, and JavaScript.

## 🎮 Features
- 8x8 grid game board
- Candy matching logic
- Score tracking
- Tile swapping

## 🚀 How to Run
1. Download the project
2. Open index.html in your browser
